# Standard Library

> ⚠️ Этот файл будет дополнен после реализации Builtins.fs.

## Арифметика

| Функция | Описание | Пример |
|---|---|---|
| `+` | сложение | `(+ 1 2)` → `3` |
| `-` | вычитание | `(- 5 3)` → `2` |
| `*` | умножение | `(* 3 4)` → `12` |
| `/` | деление | `(/ 10 2)` → `5` |
| `=` | равенство | `(= 1 1)` → `true` |
| `<` | меньше | `(< 1 2)` → `true` |
| `>` | больше | `(> 2 1)` → `true` |

## Списки

| Функция | Описание |
|---|---|
| `list` | создать список |
| `head` | первый элемент |
| `tail` | список без первого элемента |
| `cons` | добавить элемент в начало |
| `empty?` | проверить пустоту |

## Функции высшего порядка

| Функция | Описание |
|---|---|
| `map` | применить функцию к каждому элементу |
| `filter` | отфильтровать список |
| `fold` | свёртка списка |

## Maybe

| Функция | Описание |
|---|---|
| `just` | обернуть значение |
| `nothing` | пустое значение |
| `fmap` | применить функцию к Maybe |
| `bind` | монадическая цепочка |

## Ленивость

| Функция | Описание |
|---|---|
| `delay` | отложить вычисление |
| `force` | вычислить отложенное |
