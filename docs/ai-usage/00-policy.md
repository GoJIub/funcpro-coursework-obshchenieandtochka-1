# AI Usage Policy

## Общие правила

- ИИ используется как помощник: генерация черновиков, ревью, тест-кейсы.
- ИИ не является скрытым автором проекта.
- Любой код или текст, сгенерированный ИИ, проверяется человеком перед merge.
- Каждый факт использования ИИ фиксируется в логе.

## Инструменты

- ChatGPT (бесплатный / Plus)
- Claude (бесплатный)
- GitHub Copilot (если есть доступ)
- OpenAI Codex (только у AI coordinator)

## Что разрешено

- Генерировать черновики документации.
- Задавать вопросы по архитектуре.
- Просить объяснить концепцию (замыкания, letrec и т.д.).
- Генерировать тест-кейсы.
- Делать AI-review PR.

## Что запрещено

- Генерировать большие куски проекта без понимания.
- Merge AI-кода без ручной проверки владельцем зоны.
- Скрывать факт использования ИИ.

## Формат лога

Каждый участник ведёт свой файл:
- docs/ai-usage/01-log-participant-1.md
- docs/ai-usage/01-log-participant-2.md
- docs/ai-usage/01-log-participant-3.md

по следующему шаблону:

---
- Дата:
- Задача:
- Prompt:
- Ответ ИИ / краткое содержание:
- Что принято:
- Что изменено человеком:
- Связанные файлы:
- Связанный PR:
---
